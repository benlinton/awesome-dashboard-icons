# Awesome Dashboard Icons

The **most** comprehensive source of icons for dashboards, such as:

- [Homepage](https://gethomepage.dev/)
- [Homarr](https://homarr.dev/)
- [Dashy](https://dashy.to/)


## ✨ Browse the icons 

- [Icon Directory](browse/directory-a.md) - List of names with available filetypes (easier to search)
- [Icon Gallery](browse/gallery-a.md) - Mashup of icons only (faster visual scanning)


## 🚀 After updating icons

After altering icons, re-generate the static browsable files:

    ./scripts/generate_directory.py; ./scripts/generate_gallery.py
